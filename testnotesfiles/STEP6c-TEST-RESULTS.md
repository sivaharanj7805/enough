- **100% of posts have no schema at all** -- major opportunity for `missing_schema` fixes.
- **Schema scores are uniform (stdev=0.0)** -- likely same WordPress theme applying identical schema to all posts. This is expected for CMS-driven sites.
- **Moderate problem density (4.4 per post)** -- typical for older content not optimized for AI.
- **Most common problem: `missing_schema`** -- affects 145/145 posts (100%). This is the highest-impact fix for this site.
- **Only 1.4% of posts are AI-ready (composite ≥ 60)** -- significant optimization opportunity.
- **Strongest correlation: Citability vs Extraction (r=0.51)** -- these dimensions tend to move together.
- **Performance: 28 posts/sec** -- zero API calls, all CPU-bound regex + HTML parsing. Bottleneck is BeautifulSoup DOM construction.
